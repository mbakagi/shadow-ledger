Expected CSV columns:

sku, name, category, totalStock, buildingStock, carrierTrigger, maxCapacity, purchasingTrigger

Columns are case-insensitive. Extra columns are ignored. Missing numeric values default to 0.

Excel Import (.xlsx / .xls):

The first sheet will be read. The first row must contain column headers matching the same names as CSV.

Supports files exported from BS Comm, SAP, or any ERP with standard column names.

JSON Import:

[{"sku":"WH-001","name":"M8 Bolt","totalStock":100, ...}]

Expects an array of objects. Each object should have sku and/or name fields.

TSV (Tab-Separated Values):

Same as CSV but uses tabs as delimiters. Common when pasting from spreadsheets or clipboard.
